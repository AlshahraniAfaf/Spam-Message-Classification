# Spam Message Classification Project

A Machine Learning project focused on classifying text messages into "Spam" or "Ham" using Natural Language Processing (NLP) techniques.

## Project Steps
1. *Data Cleaning & Text Preprocessing*: Cleaned text by removing punctuation and converting to lowercase.
2. *Feature Extraction*: Transformed text data into numerical vectors using TF-IDF Vectorizer.
3. *Model Training*: Evaluated multiple classification models including Naive Bayes, Logistic Regression, Linear SVM, Decision Tree, and Random Forest.
4. *Evaluation: The **Decision Tree* model achieved the highest accuracy of *92.3%*.

## Tools Used
* Python
* Pandas & NumPy
* Scikit-learn
* Matplotlib & Seaborn
